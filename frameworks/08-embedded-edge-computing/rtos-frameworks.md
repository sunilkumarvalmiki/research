# Real-Time Operating Systems (RTOS)

## Major RTOS

**FreeRTOS**: AWS-backed (40+ architectures, 10KB footprint, MIT license, IoT)
**Zephyr**: Linux Foundation (400+ boards, modern architecture, Apache 2.0)
**NuttX**: POSIX-compliant (Apache project, scalable, real-time)
**RT-Thread**: Chinese ecosystem (IoT-focused, GUI support, growing)

**Recommendation**: FreeRTOS for AWS IoT, Zephyr for modern embedded development.
