# Edge ML Deployment

## Major Frameworks

**TensorFlow Lite**: Mobile/embedded standard (100KB runtime, quantization, Google-backed)
**ONNX Runtime**: Cross-framework (optimized inference, Microsoft, multi-platform)
**Edge Impulse**: Full ML pipeline (AutoML, deployment tools, embedded-focused)
**TinyML**: Ultra-low power (microcontroller ML, research area)

**Recommendation**: TensorFlow Lite + Edge Impulse for most edge ML deployments.
