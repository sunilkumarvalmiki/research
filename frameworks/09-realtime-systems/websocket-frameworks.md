# WebSocket Frameworks

## Major Frameworks

**Socket.io**: Feature-rich (fallback mechanisms, rooms, 10K+ connections/instance)
**ws**: Minimal (Node.js, performant, 100K+ connections possible, bare-bones)
**Soketi**: Open-source Pusher (scaling-friendly, compatible API, cloud-native)
**Ably**: Commercial (global edge network, guaranteed delivery, expensive)

**Recommendation**: Socket.io for features, ws for raw performance, Soketi for Pusher replacement.
