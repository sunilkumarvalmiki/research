# Batch Processing Frameworks

## Major Frameworks

**Apache Spark**: Industry standard (100GB+ datasets, mature ecosystem, PySpark/Scala)
**Polars**: Rust-based (5-10x faster than Pandas, single-machine, Python/Rust)
**DuckDB**: In-process SQL (analytical queries, OLAP optimized, embeddable)
**Dask**: Distributed (Pandas-like API, Python-native, scales to clusters)

**Recommendation**: Spark for large-scale distributed, Polars/DuckDB for single-machine performance.
