# Stream Processing Frameworks

## Major Frameworks

**Apache Kafka**: De facto standard (1M+ messages/sec, exactly-once semantics, mature)
**Apache Flink**: Stateful processing (event time, complex CEP, SQL support)
**Redpanda**: Kafka-compatible (10x faster, no JVM/Zookeeper, simpler ops)
**Apache Pulsar**: Multi-tenancy (geo-replication, unified messaging, growing)

**Recommendation**: Kafka for established needs, Redpanda for performance, Flink for complex processing.
