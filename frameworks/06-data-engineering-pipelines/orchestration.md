# Workflow Orchestration

## Major Platforms

**Airflow**: Market leader (extensive integrations, operator-based, Python DAGs)
**Prefect**: Modern Pythonic (dynamic workflows, better error handling, cloud-native)
**Dagster**: Software-defined assets (testing-first, type-safe, data quality)
**Temporal**: Workflow as code (durable execution, microservice orchestration, Go/Java/Python)

**Recommendation**: Dagster for data quality focus, Temporal for microservices, Airflow for established needs.
