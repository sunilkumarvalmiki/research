# Startup Framework Guide (1-10 Engineers)

## Core Principles

1. Speed to Market (launch in weeks)
2. Low Operational Overhead (minimal DevOps)
3. Developer Productivity (rapid iteration)
4. Generous Free Tiers (minimize costs)
5. Easy Hiring (common tech stack)

## Recommended Stack

**Full-Stack TypeScript**: Next.js + PostgreSQL + Vercel
**Alternative Python**: FastAPI + React + Railway

See parent README for complete stack specifications and cost analysis.
