# LLM Application Frameworks

## Major Frameworks

**LangChain**: Comprehensive (rapid development, abstractions, large ecosystem)
**LlamaIndex**: Data framework (RAG-focused, excellent docs, production-ready)
**Semantic Kernel**: Microsoft (C#/Python, agent orchestration, enterprise)
**DSPy**: Stanford (programmatic prompting, optimization, research-oriented)

**Recommendation**: LlamaIndex for RAG, LangChain for agents, Semantic Kernel for .NET.

**Last Updated**: December 2024
