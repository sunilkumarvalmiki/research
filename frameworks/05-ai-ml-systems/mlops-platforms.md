# MLOps Platforms

## Key Platforms

**MLflow**: Experiment tracking, model registry (framework-agnostic, open-source)
**Weights & Biases**: Best-in-class tracking (collaborative, excellent UI)
**Kubeflow**: Kubernetes-native (end-to-end ML workflows, complex)
**Metaflow**: Netflix-originated (data scientist-friendly, AWS integration)

**Recommendation**: W&B for experiments, MLflow for model registry, Kubeflow for enterprise.
