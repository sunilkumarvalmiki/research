# Frontend Framework Decision Matrix

## Quick Selection Guide

**Complex SPA**: React + Next.js
**Performance-critical**: SolidJS or Svelte  
**Content sites**: Astro or Next.js (SSG)
**Desktop apps**: Tauri (modern) or Electron (established)
**Enterprise**: React (ecosystem) or Angular (batteries-included)
