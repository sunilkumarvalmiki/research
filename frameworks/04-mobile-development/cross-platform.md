# Cross-Platform Mobile Frameworks

## Key Options

**Flutter**: Google's UI toolkit (120 FPS, 15MB apps, Dart language, hot reload)
**React Native**: Facebook's solution (JavaScript, 25MB+ apps, large community)
**Kotlin Multiplatform**: Shared logic, native UI (type-safe, Kotlin)
**.NET MAUI**: Microsoft's framework (C#, Windows/Mac/iOS/Android)

**Recommendation**: Flutter for most cross-platform needs, React Native for existing React teams.
