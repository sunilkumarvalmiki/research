# Native Mobile Development

## iOS Development

**SwiftUI**: Modern declarative UI (iOS 13+, type-safe, reactive)
**UIKit**: Traditional imperative UI (legacy apps, full control)

## Android Development

**Jetpack Compose**: Modern Kotlin UI (reactive, type-safe)
**XML Views**: Traditional approach (legacy apps)

**Recommendation**: Use modern frameworks (SwiftUI, Jetpack Compose) for new projects.
