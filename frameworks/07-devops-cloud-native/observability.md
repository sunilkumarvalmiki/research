# Observability Platforms

## Major Solutions

**Prometheus + Grafana**: Open-source standard (metrics-focused, flexible, self-hosted)
**Datadog**: Commercial leader (full-stack observability, expensive at scale, SaaS)
**OpenTelemetry**: Vendor-neutral (traces/metrics/logs standard, CNCF, instrumentation layer)
**Jaeger**: Distributed tracing (Uber-originated, CNCF graduated, open-source)
**New Relic**: APM platform (application performance monitoring, SaaS)

**Recommendation**: Prometheus + Grafana + OpenTelemetry for open-source, Datadog for commercial.
