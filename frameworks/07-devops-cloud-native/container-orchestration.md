# Container Orchestration

## Major Platforms

**Kubernetes**: Industry standard (88% adoption, complex but powerful, CNCF graduated)
**Nomad**: HashiCorp (simple deployment, multi-workload, less complex than K8s)
**Docker Swarm**: Docker native (simple clustering, declining adoption, legacy)

**Service Mesh**:
- Istio: Feature-rich, complex
- Linkerd: Lightweight, simpler
- Cilium: eBPF-based, performant

**Recommendation**: Kubernetes if scale justifies complexity, Nomad for simpler needs.
