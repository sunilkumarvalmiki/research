# DevOps & Cloud-Native Decision Matrix

## Quick Selection

**Multi-cloud IaC**: Terraform or Pulumi
**AWS-focused**: AWS CDK
**Kubernetes IaC**: Crossplane
**Container orchestration**: Kubernetes (if scale justifies)
**Observability**: Prometheus + Grafana (OSS) or Datadog (commercial)
**CI/CD**: GitHub Actions or Argo CD (GitOps)

See parent README for detailed comparison and cost analysis.
