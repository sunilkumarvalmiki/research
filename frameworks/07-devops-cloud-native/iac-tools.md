# Infrastructure as Code Tools

## Major Tools

**Terraform**: Multi-cloud leader (HCL language, 25K+ providers, mature state management)
**Pulumi**: Real languages (TypeScript/Python/Go, better testing, cloud-native)
**AWS CDK**: AWS-optimized (TypeScript/Python, CloudFormation-backed, type-safe)
**Crossplane**: Kubernetes-native (GitOps-friendly, API-driven, declarative)

**Recommendation**: Terraform for multi-cloud, Pulumi for better DX, AWS CDK for AWS-only.
